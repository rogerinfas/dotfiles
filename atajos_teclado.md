# ⌨️ Atajos de Teclado (Hyprland)

El gestor de ventanas (Hyprland) utiliza la tecla **SUPER** (la tecla de Windows) como modificador principal para casi todas las acciones.

Si necesitas restaurar esta configuración exacta en otra computadora, simplemente copia el archivo adjunto `keybindings_backup.lua` en la ruta `~/.config/hypr/conf/keybindings/default.lua`.

---

## 🚀 Aplicaciones Principales
| Atajo | Acción |
|---|---|
| **SUPER + Enter** | Abrir la Terminal (Kitty) |
| **SUPER + B** | Abrir el Navegador Web |
| **SUPER + E** | Abrir el Explorador de Archivos (Thunar) |
| **SUPER + D** | Abrir el Lanzador de Aplicaciones (Rofi) |

## ⚙️ Menús y Utilidades del Sistema
| Atajo | Acción |
|---|---|
| **SUPER + S** | Búsqueda Web rápida (Rofi) |
| **SUPER + CTRL + S** | Cambiador de Ventanas abiertas (Rofi) |
| **SUPER + ALT + V** | Gestor del Portapapeles (Cliphist) |
| **SUPER + ALT + E** | Selector de Emojis |
| **SUPER + ALT + C** | Calculadora |
| **SUPER + SHIFT + N** | Panel de Notificaciones (SwayNC) |
| **SUPER + SHIFT + E** | Menú de Ajustes Rápidos (Sidebar) |
| **CTRL + ALT + P** | Menú de Energía (Apagar/Reiniciar) |
| **CTRL + ALT + L** / **SUPER + L** | Bloquear la pantalla (Hyprlock) |
| **SUPER + H** | Ver todos los atajos de teclado en pantalla |

## 📸 Capturas de Pantalla (Screenshots)
| Atajo | Acción |
|---|---|
| **SUPER + Impr Pant** | Capturar toda la pantalla al instante |
| **SUPER + SHIFT + S** | Seleccionar un área para capturar (y copiar al portapapeles) |

## 🪟 Gestión de Ventanas y Layouts
| Atajo | Acción |
|---|---|
| **SUPER + Q** | Cerrar la ventana activa |
| **SUPER + SHIFT + Q** | Forzar cierre (matar proceso) de la ventana activa |
| **SUPER + Flechas (← ↑ → ↓)** | Cambiar el foco (moverse) a otra ventana |
| **SUPER + CTRL + Flechas** | Mover la ventana activa de lugar |
| **SUPER + ALT + Flechas** | Intercambiar la ventana activa con otra |
| **SUPER + SHIFT + Flechas** | Redimensionar (agrandar/achicar) la ventana activa |
| **SUPER + Espacio** | Alternar ventana entre flotante / mosaico |
| **SUPER + SHIFT + F** | Pantalla completa (Fullscreen) |
| **SUPER + CTRL + F** | Maximizar ventana (manteniendo la barra visible) |

## 🖥️ Escritorios Virtuales (Workspaces)
| Atajo | Acción |
|---|---|
| **SUPER + [1-0]** | Cambiar al escritorio 1, 2, 3... 10 |
| **SUPER + SHIFT + [1-0]** | Mover la ventana activa al escritorio 1, 2, 3... |
| **SUPER + Rueda del Ratón** | Navegar rápidamente entre escritorios |
| **SUPER + U** | Abrir/Ocultar el escritorio especial oculto ("Scratchpad") |
| **SUPER + SHIFT + U** | Enviar una ventana al escritorio especial oculto |

## 🔄 Recargas y Efectos
| Atajo | Acción |
|---|---|
| **SUPER + W** | Selector de Fondo de Pantalla (App) |
| **SUPER + SHIFT + W** | Aplicar un efecto aleatorio al fondo actual |
| **SUPER + N** | Alternar filtro de luz nocturna (Night Light) |
| **SUPER + ALT + R** | Recargar la barra de estado (Waybar) |
| **SUPER + CTRL + R** | Recargar la configuración completa de Hyprland |
| **CTRL + ALT + Supr** | Salir completamente de Hyprland (Cerrar Sesión) |

---
*Nota: También están configuradas las teclas multimedia estándar de tu teclado (Subir/Bajar Volumen, Mutear, Brillo, Play/Pause, Siguiente/Anterior).*
